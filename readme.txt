== INSTALLATION AND UPGRADE==

  Execute the install.php - for upgrading it will detect which version you are using and execute the appropriate files.
  DELETE update1.x-2.0.php, update2_2_1.php, update2_2_2.php, update2_2_3.php,update2_2_4.php and install.php, they are a security risk
